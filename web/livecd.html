Rejoin ISO (Linux/macOS)

Open terminal in the download directory and run:

cat OROBOROS-OS-V1.0-Live.part_* > OROBOROS-OS-V1.0-Live.iso

Verify with: sha256sum OROBOROS-OS-V1.0-Live.iso

Rejoin ISO (Windows)

Open Command Prompt in the download directory and run:

copy /b OROBOROS-OS-V1.0-Live.part_aa + OROBOROS-OS-V1.0-Live.part_ab + OROBOROS-OS-V1.0-Live.part_ac + OROBOROS-OS-V1.0-Live.part_ad + OROBOROS-OS-V1.0-Live.part_ae OROBOROS-OS-V1.0-Live.iso

Create Bootable USB

Flash the ISO to a USB drive (32 GB or larger recommended)

Windows - Rufus

  1. Download Rufus from rufus.ie
  2. Insert USB drive (32 GB+)
  3. Select the ISO file
  4. Click "Start" to flash
  5. Boot from USB (F12 at startup)

Windows/macOS/Linux - balenaEtcher

  1. Download from etcher.balena.io
  2. Flash from URL or file
  3. Select the ISO
  4. Select USB drive
  5. Click "Flash"

Linux - dd Command

  1. Identify USB device: lsblk
  2. Flash ISO: sudo dd if=OROBOROS-OS-V1.0-Live.iso of=/dev/sdX bs=4M status=progress
  3. Sync: sync
  4. Boot from USB

Virtual Machine

  1. Create new VM in VirtualBox/VMware
  2. Set RAM: 8 GB minimum (16 GB recommended)
  3. Set storage: 50 GB minimum
  4. Mount ISO as CD/DVD
  5. Boot and select "Live System"

System Requirements

Minimum and recommended specifications for running OROBOROS OS Live CD

Minimum

  • CPU: 4 cores (x86_64)
  • RAM: 8 GB
  • GPU: Any graphics card
  • Storage: 50 GB (for persistent)
  • USB: 32 GB

Recommended

  • CPU: 8+ cores (x86_64)
  • RAM: 16+ GB
  • GPU: NVIDIA RTX 3060+ (for local AI)
  • Storage: 100+ GB SSD
  • USB: 64 GB USB 3.0+

Boot Support

  • BIOS (Legacy) boot
  • UEFI boot
  • Secure Boot (may need to disable)

Default Login

  • Username: oroboros
  • Password: (none - auto login)
  • Root: sudo enabled
